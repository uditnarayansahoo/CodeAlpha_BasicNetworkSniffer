# Basic Network Sniffer

## Description
A Python-based network sniffer developed using Scapy to capture and analyze network traffic packets in real time.

## Features
- Capture live network packets
- Display source and destination IP addresses
- Detect TCP, UDP, and ICMP protocols
- Analyze packet payloads
- Monitor network traffic

## Technologies Used
- Python
- Scapy

## Installation

pip install scapy

## Run

python sniffer.py
